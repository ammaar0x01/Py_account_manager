### Installation
*PIP (Pip Installs Packages / Pip Installs Python / preferred installer program)*

Install pip
```
python get-pip.py
```

Install Python modules from requirements.txt using pip
```
pip install -m requirements.txt
```

View all installed Python packages
```
pip list
```
